# qrshot
Decode QR code on screen, or encode a string then echo QR code on terminal for MacOS.

## Usage

### Decode

Run `qrshot` in terminal, use mouse to select QR code area on screen.

### Encode

```
$ qrshot "Sample string"
```
![encoding screenshot](encode.png)

## Dependencies

- screencapture
- zbarimg
- qrcode-terminal
